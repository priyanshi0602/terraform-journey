# Day 1 - Introduction to Terraform

## What is Terraform?

Terraform is an Infrastructure as Code (IaC) tool developed by HashiCorp.

It helps automate infrastructure creation using code instead of manually creating resources in cloud consoles.

## Why Terraform?

- Automation
- Consistency
- Version Control
- Multi-cloud Support

## Key Terms

### IaC
Infrastructure as Code.

### HCL
HashiCorp Configuration Language used by Terraform.

## Important Commands

```bash
terraform version
terraform init
terraform plan
terraform apply
terraform destroy
